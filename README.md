# frontend_base_ver06

## Overview
gulp + webpackによる静的ファイル生成のための開発環境テンプレートです。

## Requirements
- node.js v10.15.1で動作確認
- gulp v3.9.1で動作確認  

※その他の開発に必要なpackageのバージョンはpackage.jsonを参照

## Description


## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost
$ gulp default

# build for production
$ gulp publish
```
